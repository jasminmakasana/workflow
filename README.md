# react-wizard-viewer

## Summary

React Workflow viewer

## Key Features

- Visualize the Workflow
- Multistep Forms
- Form Validation
- Works with React.js

## Example

```
import { Workflow } from "react-wizard-viewer";

class App extends React.Component {
  constructor(props) {
    super(props);
    this.state = {};
  }
  render() {
    return (
      <div>
        <Workflow />
      </div>
    );
  }
}

export default App;
```
